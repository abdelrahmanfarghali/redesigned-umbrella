# Git Course Page

A simple, clean HTML page for a Git course with a minimalist white design.

## Files

- `index.html` - Main HTML file
- `styles.css` - Stylesheet (link in HTML head)

## Quick Start

1. Add this line to your HTML `<head>`:
```html
<link rel="stylesheet" href="styles.css">
```

2. Make sure both files are in the same directory
3. Open `index.html` in your browser

## Features

- Clean white aesthetic
- Responsive design
- Professional styling
- Easy to customize
